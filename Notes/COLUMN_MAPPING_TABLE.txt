HOW TO USE THIS MAPPING:

1. IDENTIFY WHAT YOU HAVE:
   ✓ Row = 1 patient sample
   ✓ Columns = different features/measurements
   ✓ All values are standardized (mean=0, std=1)

2. CHOOSE YOUR FEATURES:
   For Machine Learning Models:  Use PCA versions (4 columns, cleaner)
   For Correlation Analysis:     Use original versions (interpretable)
   For Feature Importance:       Use GA selected versions (best predictors)

3. EXAMPLE ANALYSIS:
   Question: "Does high mutation count → worse survival?"
   Use: total_mutations_dna vs overall_survival_clinical
   Calculate: correlation coefficient

4. WHAT'S IN EACH COLUMN:
   Clinical Features = Patient characteristics
   DNA Features = Tumor genetic characteristics  
   IDs = Patient/sample identifiers
   Transformed = Mathematical combinations
